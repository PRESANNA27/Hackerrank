A librarian needs to arrange books on a shelf in ascending order based on their page count. Each book has a unique page count, and the librarian wants to use the Bubble Sort algorithm to sort them efficiently.

Given an unsorted list of book page counts, help the librarian sort the books in ascending order using Bubble Sort.

Input Format

An integer n representing the number of books.
A list of n space-separated integers, where each integer represents the number of pages in a book.
Constraints

NA

Output Format

A single line of n space-separated integers representing the sorted page counts in ascending order.

Sample Input 0

5
45 12 89 34 22
Sample Output 0

12 22 34 45 89
Explanation 0

The given list [45, 12, 89, 34, 22] is sorted step by step using Bubble Sort, resulting in [12, 22, 34, 45, 89].

Sample Input 1

4
100 50 25 75
Sample Output 1

25 50 75 100
Explanation 1

After applying Bubble Sort, the sorted order is [25, 50, 75, 100].
