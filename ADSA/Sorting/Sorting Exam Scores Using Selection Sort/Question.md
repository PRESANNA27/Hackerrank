A school conducted an examination for multiple students, and the examiner needs to arrange the scores in ascending order to determine rankings. The examiner decides to use Selection Sort to efficiently sort the scores.

Given a list of exam scores, write a program to sort them in ascending order using Selection Sort.

Input Format

An integer n representing the number of students.
A list of n space-separated integers, where each integer represents a student's exam score.
Constraints

NA

Output Format

A single line of n space-separated integers representing the sorted scores in ascending order.

Sample Input 0

5
45 23 89 12 67
Sample Output 0

12 23 45 67 89
Explanation 0

The given list [45, 23, 89, 12, 67] is sorted using Selection Sort, resulting in [12, 23, 45, 67, 89].

Sample Input 1

4
78 34 56 12
Sample Output 1

12 34 56 78
Explanation 1

After applying Selection Sort, the sorted order is [12, 34, 56, 78].
